Love-Terminal
========

Simples funcionalidade de inciar o codigo pelo terminal, sem a necessidade de colocar a pasta do codigo.

Somente para linux.

Instalação
========
	sudo cp love-terminal /bin/
Simples, não?

Utilização
========
	cd /home/user/PastaDoCodigo/
	love-terminal
	
ou para arquivos .love

	cd /home/user/PastaDoCodigo/
	love-terminal -f MeuCodigo
